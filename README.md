# TriSolve

TriSolve is a platform that connects problem creators, solution creators, and investors. It aims to foster innovation by creating a marketplace for ideas and funding.

## Core Features

-   **Problem & Solution Marketplace**: Users can submit real-world problems and propose innovative solutions.
-   **Investor Hub**: Investors can browse problems and solutions to find their next venture.
-   **Reputation System**: A public voting and review system helps establish credibility for all users.
-   **AI-Powered Matchmaking**: Leveraging generative AI, investors receive personalized suggestions for high-potential problem/solution creator pairings based on their profile and community reputation data.

## Tech Stack

-   **Framework**: [Next.js](https://nextjs.org/) (with App Router)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
-   **AI**: [Google's Gemini model via Genkit](https://firebase.google.com/docs/genkit)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start by exploring the **AI Matchmaking** tab on the homepage to see the generative AI feature in action.
